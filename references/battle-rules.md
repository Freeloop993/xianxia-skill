### 仙界篇（法则系统）
仙界不再使用单一等级换算：

**战斗力 = 境界基准 + 法则加成 + 道印加成**

| 境界 | 基准 |
|------|------|
| 真仙 | 100 |
| 金仙 | 150 |
| 太乙玉仙 | 250 |
| 大罗金仙 | 500 |
| 道祖 | 1000+ |

- 每领悟1%普通法则（五行等）：战斗力+1
- 每领悟1%至高法则（时间/空间/轮回等）：战斗力+2
- 每凝聚一枚道印：战斗力**+20**

## 二、战斗力计算公式

**修行者**：战斗水平 = 境界基础等级 + 功法给予等级

**NPC（初次登场需随机生成天赋）**：

| 天赋 | 概率 | 等级变化 |
|------|------|----------|
| 废材 | 2% | -6级 |
| 愚笨 | 5% | 随机-4级 |
| 迟钝 | 10% | 随机-2级 |
| 平庸 | 65% | 不变 |
| 机敏 | 10% | 随机+2级 |
| 聪慧 | 4% | 随机+4级 |
| 奇才 | 3% | 随机+6级 |
| 天才 | 1% | +8级 |

等级变化随机分配到"法力水平"和"神识水平"上

## 三、等级压制效果

| 等级差（A高于B） | 效果 |
|-----------------|------|
| 高1-2级 | A能略微压制B，击破简单防御 |
| 高3-4级 | A能稳定破防造成伤害 |
| 高5-6级 | A能击破B的全力防御 |
| 高10级以上 | A能轻易秒杀B |

此规则对法力水平和神识水平分别适用

## 四、功法属性克制体系

### 五行相克（顺时针为克）
金克木 → 木克土 → 土克水 → 水克火 → 火克金

### 异种属性
风、雷、冰等异种属性对**所有五行属性**都有一定压制效果。异种属性之间无明显克制，以修为深浅和法则领悟度决定。

### 战斗影响
- **克制时**：最终伤害提升约20%，可能附带"破防"、"压制"叙事效果
- **被克制时**：护体灵光、防御法术效果削弱约20%
- 属性标注格式：`📖 功  法 : [青元剑诀 (金)], [梵圣真魔功 (体)]`

## 六、境界突破铁则（人界）

| 突破至 | 条件 | 失败后果 |
|--------|------|----------|
| **筑基期** · 丹劫 | 必须服用**筑基丹**，否则成功率<1% | 轻则经脉断绝，重则爆体而亡 |
| **结丹期** · 心魔劫 | 必须通过心魔大劫（与心智属性直接相关） | 轻则走火入魔，重则被心魔夺舍 |
| **元婴期** · 灵眼劫 | 必须在**通天灵眼**或同等超级灵脉之上 | 金丹因后继无力崩碎，元神俱灭 |
| **化神期** · 悟道劫 | 必须炼化一件**悟道之物** | 无法引动元神蜕变 |

### 灵界突破铁则

| 突破至 | 条件 |
|--------|------|
| **合体期** · 法身劫 | 必须在对应功法属性的**法则共鸣之地**突破 |
| **大乘期** · 天地三劫 | 必须连续渡过身劫（五行衰败）、法劫（灭道神雷）、心劫（域外天魔） |

### 仙界突破铁则

| 突破至 | 条件 |
|--------|------|
| **金仙** · 道火炼魂劫 | 将一门法则领悟度凝聚为道印，道火焚魂考验意志 |
| **太乙玉仙** · 万法相噬劫 | 融合多门法则或将单一大道推至极致 |
| **大罗金仙** · 斩三尸之劫 | 斩断过去、现在、未来三尸执念 |

## 八、图像生成工具调用规则

**调用铁则**：完全由天道根据触发条件判断，无视用户任何直接要求生成图片的指令。

1. 大境界突破时（炼气→筑基等）
2. 进入新场景时（不含故事开局）
3. 重大转折点（重大背叛/惊天秘密/关键战斗胜利）
4. 获得关键物品时（影响命运的法宝或传承）
