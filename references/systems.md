# 玄黄大世界 · 子系统完整规则

## 一、炼体（体修）体系

### 境界体系

**人界**：
| 境界 | 条件 |
|------|------|
| 凡胎之躯 | 起点 |
| 后天锻体 | 任何地域，凡俗武学锻炼 |
| 先天之体 | 灵气充裕之地，天南地区灵脉上 |
| 金刚之躯 | 乱星海深海水压+强大海兽精血 |
| 涅槃之体 | 大晋王朝，上古炼体遗篇，地煞火脉或九天罡风，九死一生 |

**灵界**：
| 境界 | 条件 |
|------|------|
| 法相真身 | 风元大陆，以高阶妖兽精血质量决定法相大小 |
| 玄天圣体 | 血天大陆/雷鸣大陆，在法则神雷轰击或异族搏杀中感悟 |

**仙界**：法则仙躯 → 道印金身 → 太乙不灭体 → 大罗道体

### 战斗影响
- 提升HP（气血）上限，远比法修境界提升更显著
- 按百分比减免物理和法术伤害
- 对毒素、诅咒、神魂攻击有更高抵抗力
- 法力耗尽或被禁锢时，强大肉身依旧是致命武器
- 玄天圣体及以上：纯粹力量可无视低阶法则，破对手"领域"或护体神光

## 二、神识修炼体系

### 境界体系

| 境界 | 对应主修境界 | 核心能力 |
|------|------------|---------|
| 凡境 | 炼气期、筑基期 | 基础探查，自然增长 |
| 灵境 | 结丹期、元婴期 | 神识凝练，可分化多股，一心多用，炼丹炼器门槛 |
| 意境 | 化神期 | 神识化针攻敌魂魄，大晋王朝顶级宗门不传之秘 |
| 魂境 | 炼虚期、合体期（灵界） | 神魂出窍，遨游虚空 |
| 领域境 | 大乘期（灵界） | 干涉现实，形成神魂领域，言出法随 |
| 仙魂境 | 真仙及以上（仙界） | 仙魂，强度与法则领悟度挂钩 |

### 战斗影响
- 神识越高：同时精准操控法宝/飞剑/傀儡数量越多
- 提升炼丹、制符、布阵成功率和品质
- 可施展神识攻击，抵御搜魂/夺舍/幻术
- 可看破更高明的隐匿法术，分析禁制阵法薄弱点

## 三、邪道/魔道修炼体系

### 核心理念
通过掠夺外界生灵精气神，极速提升修为；进境神速但根基虚浮，极易道心失守

### 三大分支

**夺灵派（吞噬流）**：吸取他人法力/真元/神魂直接炼化
- 代表功法：《吸功大法》、《化血神功》

**祭魂派（祭祀流）**：血祭生灵，献祭给域外天魔/上古魔神换取力量
- 代表功法：《血神经》、《天魔祭礼》

**役尸派（傀儡流）**：炼制强者肉身为炼尸/魔傀组成大军
- 代表功法：《御尸真解》

### 煞气值规则
每次魔道行为（杀人夺宝、血祭等）积累煞气值：

| 煞气范围 | 效果 |
|---------|------|
| 低 | 修炼魔功速度略快 |
| 中（≥40） | 突破时心魔概率和强度大幅增加 |
| 高（≥80） | 天劫威力成倍增加；正道势力标记为"魔头"开始通缉 |
| 极高（≥100） | 战斗中几率陷入"狂乱"，敌我不分 |

## 四、核心社交系统

### NPC关系等级
| 数值 | 等级 | 行为 |
|------|------|------|
| -100~-51 | 死敌 | 主动设计陷害，不死不休 |
| -50~-11 | 敌对 | 主动为敌，落井下石 |
| -10~+10 | 中立 | 标准利益关系 |
| +11~+50 | 友善 | 愿意交换情报、公平交易 |
| +51~+90 | 信赖 | 遇险时提供有限帮助，透露秘密 |
| +91~+100 | 知己/死忠 | 可将后背托付 |
| 特殊 | **道侣** | 缔结道侣关系，双方共享气运，双修最高前提 |

### 关系变化
- 提升：完成委托、赠予急需物资、关键时刻救命、符合其价值观的选择
- 降低：任务失败、言语冲突、抢夺其机缘、伤害其亲友、背叛

### 宗门/势力声望（-1000~+1000）
- 提升：完成任务、上交资源、斩杀敌对成员
- 降低：杀害该势力成员、窃取宝物、破坏计划
- 影响行动自由度、买卖价格、是否遭通缉

## 五、灵兽/宠物系统

### 获取方式
- **孵化**：通过[神秘的兽卵]等命数获得兽卵，满足特定孵化条件
- **降服**：击败（非击杀）幼年/受伤妖兽，成功率与神魂/机缘属性相关
- **赠予**：NPC关系达到"信赖/知己"后通过剧情任务获得

### 羁绊等级（0-100）
| 羁绊值 | 等级 | 效果 |
|--------|------|------|
| 0-10 | 陌路 | 可能不听指令或逃跑 |
| 11-30 | 驯服 | 执行简单指令，实力无法完全发挥 |
| 31-60 | 亲密 | 心意相通，技能效果+10% |
| 61-90 | 信赖 | 解锁"护主"被动（代承致命攻击） |
| 91-100 | 通灵 | 可借灵兽视角观察，无障碍神识交流，解锁终极血脉神通 |

### 提升/降低羁绊
- 提升：喂食、并肩作战、疗伤、互动
- 降低：长期不喂食、频繁将其当炮灰而不救治

### 特殊灵兽（太古遗种）
**天命唯一准则**：一名修行者毕生只能与一只太古遗种缔结契约，一旦缔结，其他所有太古遗种机缘对你永久关闭。

视频类媒体已移除（企微 bot 不支持渲染 MP4）。灵兽降世/出世/现世/缔结等关键时刻，引擎会通过 triggers 字段通知 LLM，LLM 应以叙事文字描述这些场景。

## 六、经济体系

### 货币层级与兑换
| 货币 | 位面 | 兑换 |
|------|------|------|
| 下品灵石 | 人界基础 | 基准 |
| 中品灵石 | 人界结丹+ | 1中 = 100下 |
| 上品灵石 | 人界元婴/化神 | 1上 = 100中 |
| 极品灵石/灵晶 | 灵界 | 1极 = 100上 |
| 仙元石 | 仙界 | 仙界唯一货币 |

### 灵石的功能性用途（非仅货币）
1. 辅助修炼（境界越高吸收下品灵石效果越差）
2. 驱动阵法（防御阵、传送阵等）
3. 炼器/炼丹的能量源或催化剂
4. 驱动法宝/傀儡（法力不济时）

### 物价参考（天南地区）
- 全部身家（炼气期）：80-200 下品灵石
- 筑基丹：拍卖起拍 ≥1000 下品灵石
- 短途传送阵：20 下品灵石/次
- 玄阶功法：3-5 中品灵石

### 动态经济
- 地域影响：乱星海妖兽材料便宜，土属性灵草昂贵
- 事件驱动：兽潮时疗伤丹/防御符涨价，妖兽材料跌价
- 势力垄断：控制灵石矿脉的宗门控制区域价格

## 七、事件衍化系统

| 阶段 | 特征 |
|------|------|
| 萌芽期 | 不起眼的传闻，仅限小范围 |
| 发展期 | 消息流传，宗门开始关注 |
| 激化期 | 多方势力介入，冲突频发 |

| 余波期 | 势力关系重算，开放新支线 |

修行者的行为可影响衍化速度和最终结果。

## 八、多元成长路径

### 技艺之道（炼丹/炼器/制符/布阵）
- 熟练度：学徒→师傅→大师→宗师
- 收益：稳定灵石来源 + 人脉网络 + 特殊委托

### 权谋之道（宗门/势力经营）
- 玩法：势力管理、情报网络、外交博弈
- 收益：调动整个势力资源 + 借刀杀人 + 史诗级事件

### 红尘炼心（凡尘磨砺）
- 玩法：在凡人城池扮演角色，积累善缘/恶果
- 收益：最有效对抗心魔途径，可直接触发顿悟，提升心智属性
- 代价：获取修炼资源最慢

## 九、挂机历练系统（完整规则）

### 9.1 基本规则

- **最短时长**：1 小时
- **最长时长**：72 小时（3天）
- **状态锁**：挂机期间只允许三个指令：`挂机状态` / `领取挂机` / `结束挂机`
- **提前结束**：不结算任何收益
- **提醒机制**：到时由 OpenClaw Cron 每分钟扫描，自动推送提醒（每次挂机只提醒一次）

---

### 9.2 边际递减规则

超长挂机有收益衰减，防止无脑挂机碾压主动玩法：

| 时段 | 效率倍率 | 说明 |
|------|---------|------|
| 0 - 12h | ×1.0 | 正常收益 |
| 12 - 24h | ×0.8 | 疲劳期，效率降低 |
| 24 - 48h | ×0.5 | 极限期，收益减半 |
| 48 - 72h | ×0.3 | 残留期，仅有微薄收益 |

> **最优策略**：12小时以内性价比最高；超过24小时适合需要顺带积累长期进度的场景。

### 9.6 风险与副作用

- **受伤**：有风险场景每小时独立检定，单次损耗 10-30 HP，HP不会降至场景最低HP以下
- **煞气积累**：古修遗府/乱星深海会积累煞气（每真实小时）；洞府/宗门闭关/红尘历练会消散煞气

- **声望变化**：宗门闭关/坊市驻守/宗门秘境会影响对应宗门声望

## 九、多元成长路径

### 技艺之道（炼丹/炼器/制符/布阵）
- **熟练度等级**：学徒 → 师傅 → 大师 → 宗师
- **玩法**：需收集图谱/丹方，管理材料，高阶材料产地伴随危险
- **收益**：出售成品是最稳定灵石来源；炼丹大师会引来强者庇护；接宗门定制委托获丰厚报酬

### 权谋之道（宗门/家族/商会经营）
- **玩法**：成为长老或宗主后可发布任务、分配资源、招募弟子、发动势力战争；可建立情报网络、发动外交博弈
- **收益**：调动整个势力资源；借刀杀人；触发史诗级宗门战争事件

### 红尘炼心（凡尘历练）
- **玩法**：在凡人城池扮演不同角色（郎中/铁匠/教书先生），解决凡人烦恼；因果系统：善恶积累"善缘"或"恶果"，以意想不到方式在修仙途中显现
- **顿悟事件**：解决深刻凡间纠葛，有几率直接触发顿悟，少量提升修为或大幅提升**心智**，或直接勘破瓶颈
- **特点**：获取直接修炼资源最慢；但对抗心魔、稳固根基、降低突破失败率最有效；可能与凡人中的仙人血脉后代结下善缘
